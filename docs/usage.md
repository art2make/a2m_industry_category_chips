## Автор модуля - art2make

# Использование

## Как настроить отраслевые чипсы

1. В админке откройте `Каталог -> Категории`.
2. Создайте или отредактируйте категорию, например `Отели`.
3. Откройте вкладку `Отраслевые категории`.
4. Введите название нужной категории в поле поиска.
5. Выберите категорию из выпадающего списка.
6. При необходимости задайте `Порядок сортировки`.
7. Сохраните категорию.

После сохранения выбранные категории будут показаны на странице этой категории вверху каталога.

## Пример

Для категории `Отели` можно выбрать:

- `СПА`
- `Санузлы`
- `Номерной фонд`

Для категории `Бьюти и фитнес` можно выбрать:

- `СПА`
- `Санузлы`
- `Раздевалки и душевые`

Одна и та же категория может быть показана в разных отраслевых разделах без создания дублей в дереве категорий.

## Важные особенности

- Выбранные категории являются ссылками, а не подкатегориями.
- Модуль не привязывает товары к новым категориям.
- Если у категории не выбраны отраслевые категории, блок на витрине не выводится.
- Ссылки ведут на реальные URL выбранных категорий с учетом их полного `path`.
